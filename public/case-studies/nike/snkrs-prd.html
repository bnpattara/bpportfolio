01

PROBLEM STATEMENT

Nike built a decade of dominance on artificial scarcity. Then consumers stopped playing the game.

The SNKRS app — once the engine of sneaker culture — has become a symbol of everything wrong with it: inaccessible, hostile, and designed to frustrate rather than serve. The lottery model isn't just underperforming. It is actively destroying Nike's relationship with its most valuable long-term consumers.

The "Aesthetic Individualists" — a 40% market segment — opted out of the drop model entirely. They have purchasing power, brand awareness, and latent affinity. They left because the system stopped serving them.

The real barrier isn't access — it's confidence in how to wear what you already own.
The "museum mindset" — treating shoes like stock certificates — is the actual product killing Nike's repeat purchase rate. The app is solving for scarcity when it should be solving for styling paralysis.

WHAT WE ARE NOT BUILDING

This is not a redesigned raffle UI. Not a better push notification. This is a platform pivot: from exclusivity engine to confidence infrastructure. Every feature in this document serves that singular shift.

02

STRATEGIC INSIGHT

629 research touchpoints. One non-obvious finding: the enemy isn't competition — it's paralysis.

Consumers own an average of 16+ pairs of sneakers but rotate only 2–5 regularly. The remaining 11+ pairs sit unworn — identity investments that were never activated. This is the museum mindset: shoes as collectibles rather than wearables.

The consequence is a broken replacement cycle. Nike's repeat purchase rate depends on consumers actually wearing out their shoes. The museum mindset breaks this loop. Solve for confidence, and you unlock the replacement cycle.

Solve for styling confidence → unlock the replacement cycle → recover LTV.
This is not a UX improvement. It is a business model intervention. The app becomes a styling platform that happens to sell sneakers — not a sneaker store that happens to have an app.

THE SEGMENT WE LOST

The "Aesthetic Individualist" — 40% of the 18–34 sneaker market — currently acquires identity through brands that treat them as people, not lottery tickets. New Balance, Salehe Bembury–era Puma, and Adidas Spezial are capturing this segment by default. Nike hasn't lost them because of product. Nike lost them because of the system.

Strategic repositioning: The Confidence Hub re-enters this segment's consideration set by replacing gatekeeping with guidance. The brand proposition shifts from "can you get it" to "can you wear it well."

FEATURE SET

Four features. Each addresses a discrete failure point in the current experience. P0 = MVP launch. P1 = Q2. P2 = Q3.

P0 — MVP
THE STYLE GYM
Product pages reimagined as styling resources. Every shoe ships with three complete outfit formulas — Work, Casual, Night Out — with shoppable non-Nike anchor pieces. The shoe becomes the entry point to a look, not a collectible object. Filterable by occasion, aesthetic, and body type.
Primary metric+35% avg session depth
Secondary↓ return rate (aesthetic mismatch)
ComplexityMedium — content + editorial
P0 — MVP
NIKE STYLIST AI
AI analyzes a photo of the user's existing wardrobe and recommends sneakers that integrate with what they already own. Surfaces a "Wardrobe Gap Report" — specific occasion holes, not generic suggestions. Each recommendation ships with a plain-language "Why This Works" note.
Primary metric+60% purchase confidence score
Secondary↓ 90-day return rate
ComplexityHigh — vision AI + wardrobe graph
P1 — Q2
CONFIDENCE DROPS
Replaces random raffles with 48-hour pre-order windows. Access earned through styling challenges — rewarding loyalty and creative engagement rather than luck or bot speed. Confidence Pass tier shifts queue position. Barrier moves from chance to participation.
Primary metric+40% Aesthetic Individualist capture
Secondary↑ challenge completion as NPS proxy
ComplexityMedium — challenge layer + queue logic
P2 — Q3
CONFIDENCE INDEX
Annual data-driven cultural report transforming millions of SNKRS user interactions into the definitive guide to global street style. Nike as cultural authority — not just the vendor. Distributed as editorial, press, and in-app feature. Makes Nike's aggregated data a public brand asset.

04

USER STORIES

SNKRS-001 Style Gym — Core Discovery P0
As a 26-year-old Aesthetic Individualist who stopped using SNKRS after losing three consecutive draws,
I want to see exactly how a shoe fits into a real outfit before I buy it,
so that I stop collecting sneakers I never wear and start building a wardrobe that actually works.
Each shoe product page displays minimum 3 complete outfit formulas (Work / Casual / Night Out)
Each formula includes shoppable links to every non-Nike anchor piece shown
User can filter formulas by stated aesthetic (Minimal, Streetwear, Elevated Casual)
Outfit formulas are tagged by body type / fit preference where applicable
SNKRS-002 Nike Stylist AI — Wardrobe Analysis P0
As a user with 14 pairs of sneakers I barely rotate,
I want the app to analyze what I already own and tell me specifically what would fill a genuine gap in my wardrobe,
so that my next purchase feels like an intelligent decision — not another impulse I'll regret.
User uploads photos of existing sneakers; AI identifies silhouette, colorway, and wear context
AI generates a "Wardrobe Gap Report" identifying missing occasions with specifics
App surfaces recommendations ranked by gap-fill score, not inventory surplus
Each recommendation displays a "Why This Works" explanation in plain language
User receives push alerts only when a high gap-fill score shoe drops
SNKRS-003 Confidence Drops — Participation Access P1
As a sneaker enthusiast who has lost 12 consecutive raffles and given up on SNKRS drops,
I want access to limited drops to be tied to how engaged I am — not random chance,
so that my loyalty and participation actually change my outcome.
Each high-demand drop displays an optional 48-hour styling challenge in advance
Completing a challenge earns a Confidence Pass granting queue priority (not guaranteed)
Confidence Pass tier is visible in user profile — gamified but not gameable by bots
Users without a Pass retain standard access; passes shift odds, not eliminate fairness
SNKRS-004 Stylist AI — Return Reduction at Checkout P0
As a Nike ops stakeholder managing a 22% post-drop return rate,
I want the purchase flow to surface AI-powered confidence checks before checkout,
so that we reduce aesthetic mismatch returns and protect margin on limited inventory.
Pre-checkout overlay shows "How This Fits Your Wardrobe" card if wardrobe graph data exists
If mismatch detected, user sees a gentle prompt (not a blocker) to review Style Gym first
Post-purchase, user receives a "First Wear" email with personalized outfit formula

09

CONSTRAINTS & ASSUMPTIONS

TECHNICAL

Nike Stylist AI requires a vision model capable of analyzing garment silhouette, colorway, and wear context from consumer-quality photos. On-device processing preferred for privacy compliance. Wardrobe graph requires persistent user identity — SNKRS account required for all AI features.

BUSINESS

Confidence Drops cannot fully replace SNKRS Draw in Year 1. Nike's highest-hype releases (Jordan 1 retros, collab drops) generate media coverage value from lottery mechanics. Confidence Drops introduced on mid-tier and general-release SKUs first, with premium migration gated on performance data.

Aesthetic Individualists will re-engage if access model changes High 629 research touchpoints confirm willingness; A/B on first release
