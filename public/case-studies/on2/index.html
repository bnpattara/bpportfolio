01The cultural moment

The best hardware in running
ships with no software at all.

Running became the last truly personal sport. No team. No coach. No crowd. Just a body, a road, and a decision to move. The tools runners chose reflected that: premium, precise, engineered with obsessive care.

Then they came home, opened an app, and handed their data to someone else.

02The brand

On built the most precise shoe in running.
Then stopped.

On Running’s CloudTec sole geometry, Helion foam, and liquid-injected Speedboard plate turned every footstrike into a calibrated event. Runners chose On because it felt like a precision instrument.

At the digital layer, that relationship ended at the register. On had zero owned software surface area. The default tracker was Nike Run Club. The default community was Strava. On made the shoe. Nike owned the runner.

03The reframe

On runners aren’t chasing leaderboards.
They’re chasing themselves.

Nike Run Club owns aspiration. Strava owns competition. Both platforms assume the runner’s primary motivation is social performance: to be seen, ranked, celebrated.

On runners are different. They run for self-expression: a discipline, an aesthetic, a relationship with their own body. No platform was built for that psychology. That gap was On’s moat.

Movement is the act. The shoe is the object. The platform is the witness.

04The idea

Apex: Software built to the same standard
as the shoe.

Not a fitness tracker. Not a social network. A silent, precision instrument that knows exactly which physical object you’re wearing and molds the digital experience around it.

The shoe is the identity unit. Apex exists to honor it.

05The system

Every design decision was a rejection
of something familiar.

The molecular design system enforced the same restraint as On’s physical hardware. Nothing decorative. Nothing arbitrary. Every rule tied to a functional need.

  • Split mode (Light mode for browsing. Dark mode the instant you hit run. The transition is the feature) a physiological trigger that signals you’re in the work.
  • Volt rule (One accent color. Maximum three instances per screen. Scarcity creates authority) the eye goes exactly where it should.
  • Zero radius (No rounded corners anywhere in the interface. Not approachable consumer tech) a serious tool. The visual language of industrial instruments.
  • DM Mono (Monospaced font for all live data. Every digit occupies identical horizontal space. The stopwatch never jitters. Precision is felt, not just seen.
  • No likes) Five On-native reactions replace the heart: Push, Solid, Pace, Zone, Respect. Each requires the viewer to have actually looked at the run. Cognitive friction replaces cheap dopamine.
  • Shoe garage (Automatic odometer per registered pair. At 500km, the mileage bar turns amber) not red. Reaching the end of a shoe’s lifespan is completion, not failure.

06The distribution

Phase one ships as a PWA.
Not an app. An invitation.

A native App Store launch would place Apex directly underneath sponsored ads for Nike Run Club. Instead, Apex deploys as a Progressive Web App (captured at On.com at the moment a runner drops $200 on a new pair of shoes. Maximum intent. Zero noise.

PWA infrastructure also grants full iteration velocity. Bug fixed, feature shipped, every user updated) without waiting for App Store review. The engineering team controls the clock.

07The threshold

Native infrastructure unlocks
at two numbers.

Phase Three (a full native App Store release) is not triggered by a timeline or a boardroom decision. It is triggered by evidence. The PWA must prove that anti-social-feed behavior actually retains runners in the real world.

35%
Weekly active user rate required to build native
70%
Shoe registration rate required to build native

Every deferred feature (in-app checkout, gamified challenges, native push notifications) was a documented tradeoff. The core philosophy ships first. Everything else waits for proof.

08What changed

On stopped selling a shoe.
It started owning the runner.

Commerce as a consequence, not a purpose. A platform that protects its own integrity by refusing to become a storefront. A social feed that demands you actually look at someone’s run before you react to it.

Apex proved that software doesn’t have to be a slot machine. It can be a calibrated instrument: one that treats the digital experience with the same uncompromising standard as the physical object it was built to serve.
